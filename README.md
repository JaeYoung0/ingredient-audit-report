# 화장품 전성분 검수 리포트

Upstage **Information Extract** 로 제품 라벨에서 전성분을 추출하고,
KCIA 성분사전 표준화명칭목록(표준 성분명 21,851종 + 구명칭 4,350종)과 대조해
**표준 / 구명칭(옛 표기) / 오타 의심 / 미등록** 으로 판정한 결과 리포트입니다.

- `index.html` 한 파일로 완결 (이미지 base64 내장)
- 토큰별 문자 단위 diff, Upstage 원본 응답, 라벨 이미지 확대 포함

> 상단 정량(표준 일치율)은 참고용 지표이며 정답 라벨링 기반 recall/precision 이 아닙니다.
